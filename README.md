# kilian.sh

Java, Spring, 클린 아키텍처 그리고 개발 이야기를 기록하는 기술 블로그입니다.

**https://kilian.sh**

## Tech Stack

- **Framework**: React Router v7 (SSG)
- **Content**: MDX + Shiki syntax highlighting
- **Styling**: Tailwind CSS v4 + Custom CSS (Obsidian Gradient dark theme)
- **Deploy**: Vercel (static hosting)
- **Comments**: Giscus (GitHub Discussions)
- **Analytics**: Google Tag Manager + Vercel Analytics
- **SEO**: Sitemap, RSS, JSON-LD structured data

## Topics

- Domain-Driven Design / Clean Architecture
- Spring Boot / JPA / Kafka
- 성능 최적화 / 데이터 파이프라인
- 테스트 전략 / 객체지향 설계

## Development

```bash
npm install
npm run dev          # http://localhost:5173
npm run build        # Production build
npm run typecheck    # TypeScript check
npm test             # Unit tests
```
